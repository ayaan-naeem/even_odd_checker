# Even/Odd Number Checker

This is a simple C++ program that checks whether a given integer is **even** or **odd**.  

It takes a user input number and determines its parity using the modulo operator `%`.

## ðŸ§® How It Works
1. The user enters an integer.
2. The program calculates `number % 2`.
   - If the remainder is `0`, the number is **Even**.
   - Otherwise, the number is **Odd**.
3. The result is displayed on the console.

## How to run
Step 1: Compile the code

Open a terminal or command prompt, navigate to the folder containing the file even_odd_checker.cpp, and run:

g++ even_odd_checker.cpp -o even_odd_checker


This will compile the code and create an executable named even_odd_checker.

Step 2: Run the program

On the same terminal, execute:

./even_odd_checker


or on Windows:

even_odd_checker.exe
